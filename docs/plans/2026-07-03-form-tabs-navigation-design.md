# Design Doc — Form Tabs Navigation (แยกประเภทฟอร์มครุภัณฑ์และวัสดุ)

## Overview
ผู้ใช้งานระบบมีความประสงค์ต้องการแยกฟอร์มการกรอกข้อมูลสิ่งของระหว่าง **"ครุภัณฑ์"** และ **"วัสดุและอุปกรณ์"** ออกจากกัน เพื่อให้หน้าตาการกรอกข้อมูลเหมาะสมกับแต่ละประเภทสิ่งของ โดยออกแบบให้ใช้แถบเลือกแท็บ (Tabs Navigation) ในการสลับฟอร์มกรอกข้อมูล

## Proposed Changes

### 1. การสลับประเภทในฟอร์ม ([item-form.tsx](file:///d:/registry-s/features/items/components/item-form.tsx))
* เพิ่มตัวเลือกประเภทแท็บด้านบนของฟอร์ม (ครุภัณฑ์ vs วัสดุและอุปกรณ์)
* เพิ่มปุ่มสลับวิทยุเลือกระหว่าง "วัสดุสิ้นเปลือง" (`material`) และ "อุปกรณ์ทั่วไป" (`general`) ภายใต้แท็บวัสดุและอุปกรณ์
* ควบคุมการแสดงส่วนกรอกข้อมูล "ข้อมูลครุภัณฑ์ / ซีเรียล" (เลขครุภัณฑ์, Serial Number, ยี่ห้อ, รุ่น) เฉพาะตอนที่เลือกแท็บครุภัณฑ์เท่านั้น
* ในแท็บวัสดุจะซ่อนส่วนกรอกข้อมูลครุภัณฑ์เพื่อความกระชับของฟอร์ม

### 2. การกำหนดค่าเริ่มต้นและการแก้ไขข้อมูล
* เมื่อเปิดฟอร์มสร้างสิ่งของใหม่ (`new-item-sheet.tsx`): ค่าเริ่มต้นจะเป็นแท็บ **"ครุภัณฑ์"**
* เมื่อเปิดฟอร์มแก้ไขสิ่งของ (`edit/page.tsx`): ระบบจะตรวจสอบ `item.item_type`
  * หากเป็น `asset` -> แสดงแท็บ **"ครุภัณฑ์"**
  * หากเป็น `material` หรือ `general` -> แสดงแท็บ **"วัสดุและอุปกรณ์"** และเลือกปุ่มประเภทย่อยตามค่าเดิมนั้น

## UI Styling (Tailwind v4)
* แถบเลือกแท็บใช้ Tailwind Flex Layout มีเส้นขอบวิ่งด้านล่างเมื่อแอคทีฟ (Active border) และมีเอฟเฟกต์ Transition เมื่อผู้ใช้นำเมาส์ไปชี้หรือกดเลือก
